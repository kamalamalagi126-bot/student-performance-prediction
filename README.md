Student Performance Prediction

This is my beginner machine learning project.

In this project, I tried to understand how study hours, attendance and sleep hours can affect student marks. I created a small dataset and explored it using Python. I also plotted graphs to understand the data better.

After that, I trained a Linear Regression model and checked its accuracy. Through this project, I learned the basic steps of machine learning such as data exploration, visualization, model training and evaluation.

Tools used:

- Python
- Pandas
- Matplotlib
- Scikit-learn

This project was done as part of my learning process in machine learning.
